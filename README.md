# Documentación del proyecto

## Cómo funciona

Este proyecto es un clon básico de WhatsApp desarrollado con React. Permite a los usuarios autenticarse y acceder a una sala de chat donde pueden enviar y recibir mensajes simulados en tiempo real.  
El acceso al chat está protegido, es decir, solo los usuarios logueados pueden ingresar, y la sesión se gestiona utilizando "localStorage".  
Incluye una ventana emergente de configuración y una página de ayuda con información relevante sobre el funcionamiento y las tecnologías utilizadas.

## Cómo se ejecuta

1. Clonar este repositorio:
   git clone https://github.com/ValeAndrada/proyecto-final

2. Ingresar a la carpeta del proyecto:
   cd proyecto-final

3. Instalar las dependencias:
   npm install

4. Iniciar la aplicación en modo desarrollo:
   npm run dev

5. Acceder al localhost que brinda el modo desarrollo

---
**Notas:**
- Es necesario tener Node.js y npm instalados en tu computadora.
